KEY FEATURES IMPLEMENTED
=========================

✓ Event-Driven Architecture
  - Pub/sub message bus with wildcard matching
  - Event history for debugging
  - Async event dispatch

✓ Agent Orchestration
  - BaseAgent abstract class
  - Capability-based routing
  - Sequential chaining with builder
  - Parallel execution with fan-out
  - Automatic retry with exponential backoff

✓ Workflow Automation
  - DAG-based workflow definition
  - WorkflowBuilder with fluent API
  - Step retry, timeout, conditions
  - Cron and event-triggered scheduling

✓ Context Handling

USAGE EXAMPLES
==============
See SETUP.md for detailed instructions. Quick examples:

Basic Handler:
  engine = ParmEngine()
  engine.register_handler("my_handler", handler_func)
  result = engine.execute("my_handler", data)

Agent:
  agent = MyAgent()
  engine.register_agent("my_agent", agent)

Workflow:
  workflow = WorkflowBuilder("name")
    .add_step("step1", "action1")
    .add_step("step2", "action2", depends_on=["step1"])
    .build()

Context:
  resolver = ContextResolver()
  resolver.register_provider(time_provider)
  context = resolver.resolve(entity_id, entity_type)

Privacy:
  vault = DataVault(master_password="secret")
  vault.store(key, data, policy)
  result = vault.retrieve(key, accessor, level)

Events:
  engine.on_event("agent.*", callback)
  engine.emit_event(Event(...))
